# 语言要求

请始终使用简体中文回答我的所有问题和请求。

## 规则
- 所有回答都使用简体中文
- 技术术语可以保留英文，但要提供中文解释
- 代码注释使用中文
- 错误信息和提示使用中文
- 每次执行任务需要显示执行进度

## 例外
- 代码本身（变量名、函数名等）可以使用英文
- 命令行指令保持原样

# 代码要求

## 规则
- 整体软件应用系统设计需要采用高内聚、低耦合，以提升系统的可维护性和可复用性。
- 全栈的API必须符合RESTful API风格
- 必须符合模块化设计
- 添加简要的函数级的注释
- .py 的代码提交前必须进行 ruff check 和ruff format 的检查并要求检查通过
